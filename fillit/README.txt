This is the section of the 42's project "fillit"

This project goal is to create the smallest square as possible with tetriminos pieces.

How to use fillit :

Do the command "make" to create the executable and the tetriminos file

fill the file "tetriminos" with tetrimino pieces designed with some '#'
A valid pieces is composed with 4 lines of 4 characters and a '\n' followed with a '\n'.
The characters only can be '.', '#' and '\n'. Otherwise, the program will return an error.

Exemple of valid piece :
....
.###
..#.
....

use the executable named "fillit" with your tetriminos file in argument.

Have fun :)

/!/The 'fclean' command will delete the file "tetriminos"/!/
